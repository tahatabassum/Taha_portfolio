## 1. Executive Summary

This project is a high-performance, single-page, fully static developer portfolio website for **Taha Tabassum** (AI Engineer & Full-Stack Developer). It communicates technical credibility, engineering depth, and production AI application experience.

Key project traits:
- **Zero Backend / Zero Database / Static Architecture**: Completely client-side rendered, making it instantaneous to load and zero-cost to host on static edge networks like Vercel, Netlify, or GitHub Pages.
- **Dark Modern SaaS Aesthetic**: Custom-curated dark theme (Zinc `#09090b` base palette with Cyan `#06b6d4` and Indigo `#6366f1` glowing accents).
- **Engineering Case Study Focused**: Highlights real technical challenges (OpenCV chronological video frame extraction, Selenium headless web automation, Docker memory optimizations) rather than generic student projects.

## 2. Technology Stack & Implementation Details

### **Core Frameworks & Tools**
1. **React 19 (`react`, `react-dom`)**
   - Built with modern functional components.
   - Uses native React state (`useState`) for UI interaction (e.g., mobile navigation drawer toggle, active tab switches).
   - Utilizes `useEffect` for window scroll tracking and dynamic header styling.

2. **TypeScript (`typescript`)**
   - Configured in strict mode with dedicated `tsconfig.json`, `tsconfig.app.json`, and `tsconfig.node.json`.
   - All component props, data structures (projects, skill categories, engineering challenges) are typed with explicit TypeScript `interface` contracts.

3. **Vite (`vite`, `@vitejs/plugin-react`)**
   - Provides instant Hot Module Replacement (HMR) during development.
   - Bundles optimized production JavaScript and CSS chunks into `/dist` via `npm run build`.

4. **Tailwind CSS v3 (`tailwindcss`, `postcss`, `autoprefixer`)**
   - Custom Tailwind configuration (`tailwind.config.js`) extending default utility classes.
   - Defines custom font families: `font-sans` (`Inter`), `font-display` (`Outfit`).
   - Defines custom brand colors: `brand.dark` (`#09090b`), `brand.card` (`#18181b`), `brand.border` (`#27272a`), `brand.accent` (`#06b6d4`), `brand.indigo` (`#6366f1`).
   - Configured with custom keyframe animations (`fadeIn`, `slideUp`).

5. **Icons & Assets**
   - **Lucide React (`lucide-react`)**: Clean line-art icons for UI elements (`ArrowRight`, `Sparkles`, `Terminal`, `CheckCircle2`, `AlertCircle`, `ExternalLink`, `Mail`, `Phone`, `MapPin`, etc.).
   - **Inline SVGs**: Custom vectorized SVGs for GitHub and LinkedIn brand logos to avoid icon dependency mismatches.
   - **Static Assets**: Developer photo stored at `public/taha.png`.

## 7. Instructions for Master Prompt Generation (Claude / Prompt Engineer)

When using this document to prompt an LLM to redesign the UI or add new features, keep the following in mind:
1. **Preserve Content Integrity**: All real project links, email addresses, phone numbers, tech stack tags, and case study descriptions must remain accurate.
2. **Maintain Static Architecture**: Do not introduce server-side code (Node APIs, backend databases, authentication servers) unless explicitly requested.
3. **Component Modularity**: Keep code separated into clear `.tsx` files inside `src/components/`.
4. **Tailwind First**: Use Tailwind CSS utility classes and custom color variables defined in `tailwind.config.js` and `src/index.css`.
5. **Responsiveness**: Ensure all grids scale gracefully (`grid-cols-1 md:grid-cols-2 lg:grid-cols-3` or `lg:grid-cols-12`).
